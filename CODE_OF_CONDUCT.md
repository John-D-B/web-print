# Code of Conduct

**Web-Print** is a technical project. This document keeps its collaboration<br/>
&nbsp; &nbsp; spaces productive, focused, and respectful. It is short on purpose.

## Expected behaviour

- **Be respectful and professional** in all interactions.
- **Stay on topic** — keep discussion technical and relevant to the project.
- **Assume good faith**, and give clear, constructive feedback.

## Unacceptable behaviour

- Harassment, insults, or personal attacks.
- Publishing another person's private information without permission.
- Sustained off-topic disruption, spam, or trolling.

## Scope

This Code of Conduct applies within **this repository and its project spaces**<br/>
&nbsp; &nbsp; — its issues, pull requests, and discussions.<br/>
It governs conduct in those spaces only, and makes **no** claim on<br/>
&nbsp; &nbsp; participants' conduct anywhere else.

## Enforcement

The maintainer may edit, remove, or reject any contribution, and may block<br/>
&nbsp; &nbsp; any participant, for conduct that violates this Code of Conduct.<br/>
Concerns can be raised privately with the maintainer.
