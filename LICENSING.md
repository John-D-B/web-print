## The four-year conversion (Change Date)

The BSL is source-available now, not permanently restricted.<br/>
Each released version automatically converts to an open-source license —<br/>
&nbsp; &nbsp; the **GNU General Public License, version 2.0 or later** —<br/>
&nbsp; &nbsp; four years after that version is published.

In plain terms: the **current** version is always covered by the terms above,<br/>
&nbsp; &nbsp; but a version four years old becomes fully open.<br/>
This guarantees **Web-Print** can never be permanently locked away,<br/>
&nbsp; &nbsp; while the current, supported version stays commercially protected.
